Bulk Add Products via CSV

Upload a CSV file to add multiple products at once. This is ideal for setting up your initial inventory.

Your CSV file must have the following columns in this exact order:

sku, name, sale_price, cost_price, quantity

Note: The quantity column will be used to create a "Beginning Balance" journal entry for each product (Debit Inventory, Credit Opening Balance Equity).
Example CSV Format:
sku,name,sale_price,cost_price,quantity
SKU-001,Sample Product A,150.00,100.00,50
SKU-002,Sample Product B,25.50,15.00,200
SKU-003,Non-Stock Item,500.00,0.00,0
